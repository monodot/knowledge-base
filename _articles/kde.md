## My setup

- **Reduce lag in the Task Switcher (Alt+Tab):** System Settings -> Task Switcher -> deselect _"Show selected window"_. See: <https://discuss.kde.org/t/alt-tab-is-very-slow-1s-and-freezes-up-the-desktop/19756/7>

## Keyboard shortcuts

- Window Tiling:
    - Hold down `Shift` while dragging a window, to move it to a tile in your personal tiling layout.
    - `Super+T` - Opens the tiling layout editor, with options like 'Split Left/Right', 'Add Floating Tile' and 'Load Layout'. You can also set a Padding value.
